# Matrix Calculator

## Description
This project is a Python-based Matrix Calculator designed to perform linear algebra operations on square matrices of any order. It calculates inverse, adjoint, transpose, determinant, and main diagonal values using Python and NumPy.This projects highlights my skill in mathematical logic and Python programming

## Features
- Matrix inverse
- Matrix adjoint
- Determinant of a Matrix
- Matrix transpose
- Main diagonal extraction
- Supports square matrices of any order

## Technologies Used
- Python
- NumPy

## How to Run
1. Install Python
2. Install NumPy
3. Run the Python file

## Example Usage
Enter a matrix and choose an operation to perform.

## Purpose
This project was created to apply linear algebra concepts using Python programming and improve problem-solving skills.
